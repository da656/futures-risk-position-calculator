# CFFEX 期货基础参数核验说明

核对日期：2026-07-16

## 范围与来源

仅纳入中国金融期货交易所官网“产品”导航当前列示的期货：4 个股指期货（IF、IH、IC、IM）和 4 个国债期货（TS、TF、T、TL）。未纳入任何股指期权或国债期权。

所有参数均人工比对各产品页的合约表（名称、交易代码、合约乘数或合约标的、报价单位/方式、最小变动价位）。CFFEX 官网这些产品页当前公开为 HTTP URL，因此 sourceUrl 保留官网原始 HTTP 地址；Schema 的 sourceUrl 已相应改为接受绝对 URI。

## 逐行核验

### IF｜沪深300股指期货
- 来源：http://www.cffex.com.cn/cn/hs300.html
- 来源标题：沪深300股指期货 | 中国金融期货交易所
- 官网核对值：交易单位/合约标的或合约乘数为“每点300元”；报价单位/方式为“指数点”；最小变动价位为 0.2 点；交易代码为 IF。
- 每跳计算：300 × 0.2 = 60.0 元/手。
- 换算：CFFEX 产品页“合约乘数：每点300元”；0.2 点 × 300 元/点。

## 国债期货换算口径

国债期货产品页使用“百元净价报价”。故合约乘数不是名义面值本身，而是“名义面值 ÷ 100”所得的每 100 元面值单位数量：TS 为 20,000；TF、T、TL 均为 10,000。该口径使 tickValue 与合约每手实际人民币盈亏一致。
